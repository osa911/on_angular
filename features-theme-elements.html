Button Secondary

The secondary button is based on the default button. Use the modifier .bh-button-secondary for the default version or .bh-button-secondary-contrast for the contrast colour version.

Image Overlay

You can change the image by hovering the overlay by giving the overlay the class bh-overlay-image and the hovering image bh-overlay-image-hover. You can still use all default overlay combinations.

Example

Markup

<figure class="uk-overlay uk-overlay-hover">
    <img src="" alt="">
    <img class="bh-overlay-image-hover" src="" alt="">
    <figcaption class="uk-overlay-panel">...</figcaption>
</figure>
